# Krakens Canvas Site

First-pass landing page for Krakens Canvas.

Run from the project root:

```powershell
npm.cmd run site
```

Then open:

```text
http://127.0.0.1:4174/site/
```

## Stripe Donations

The donate button points to the live Stripe Payment Link:

```text
https://buy.stripe.com/9B6fZgcAodvb0wT5n4ak000
```

## Download Button

The download button points to the GitHub `v0.2.1` release ZIP. The repository must be public, or the visitor must have repository access, for the download to work.
