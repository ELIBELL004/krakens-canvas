# Krakens Canvas Site

First-pass landing page for Krakens Canvas.

Run from the project root:

```powershell
npm.cmd run site
```

Then open:

```text
http://127.0.0.1:4174/site/
```

## Stripe Donations

The donate button is intentionally a placeholder. Create a Stripe Payment Link, then replace the disabled donation link in `site/index.html`.
